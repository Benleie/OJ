## 大V理解的刷题四个境界
+ 纯刷
  + 这题我做过，但是不会
  + 面试有印象，但是挂掉
+ 硬记
  + 亲手做过（非复制粘贴）；短期内会记得，但长期会忘
  + 大多数一线大厂码农的水平
  + 只要努力，回报特别明显和直接
  + 有些人是先记住，后面才理解的，硬记是一种学习方式，并非效率低
+ 理解
  + 任何时候，拿到这道题，都可以做出来，如果忘记了，也会自己推断和分析出来。
  + 真正理解了这道题背后所隐藏的逻辑、思想和算法，并且能够扩展到类似其它问题
  + 随时可以面试
+ 掌握
  + 能把一个硬记层次的人带到理解层次

### LeetCode做完多少题，已经不是一个指标了；更关键的是，一个人“理解”了多少题。
